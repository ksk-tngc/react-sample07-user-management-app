## 概要

Chakra UI を使用した簡単な React アプリケーションです。  

以下の要素を含みます：

* TypeScript
* Atomic Design
* Chakra UI
* React Router
* axios
* カスタムフック
* Context (グローバルstate)
* 再レンダリングの最適化 (memo, useCallback)
* レスポンシブ
* JSONPlaceholder (バックエンド未実装)
* Unsplash
